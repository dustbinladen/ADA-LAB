# Lab 3: Divide and Conquer Algorithms

---

## 1. Quick Sort

- **File:** LAB3.1.cpp  
- **Aim:** Sort array using Quick Sort.  
- **Use Case:** Efficient general-purpose sorting.  
- **Algorithm:**
  1. Choose pivot.
  2. Partition array.
  3. Recursively sort subarrays.
- **Time Complexity:** O(n log n) average, O(n²) worst

---

## 2. Merge Sort

- **File:** lab3.2.cpp  
- **Aim:** Sort array using Merge Sort.  
- **Use Case:** Stable sorting algorithm.  
- **Algorithm:**
  1. Divide array into halves.
  2. Recursively sort.
  3. Merge sorted halves.
- **Time Complexity:** O(n log n)
